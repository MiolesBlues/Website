What we measure

The dataset provides performance metrics (accuracy, success rate, time), resource usage (CPU, memory), cost per task and several governance indicators (privacy, bias detection, data quality).

  • accuracy_score — correctness (0–1)
  • success_rate — success probability (0–1)
  • execution_time_seconds — speed/latency
  • cpu_usage_percent / memory_usage_mb — resources
  • cost_per_task_cents — cost at scale
  • privacy_compliance_score, bias_detection_score, data_quality_score
